def _process_slices (   self,
  slices_data,
  slices_labels,
  sample_rate 
)
private

Processes the slices by applying a Butterworth filter, extracting features, and returning the features in a dataframe.

Parameters
slices_data(list of pd.DataFrames) List of slices. Each entry is one slice/segment represented as a Dataframe.
slices_labels(list or numpy array) List of integer labels corresponding to the slices in slices_data.
sample_rate(int or float) the sampling rate of the data.
Return values
slices_data(pd.DataFrame) Dataframe with raw data of all slices. Columns represent channels (e.g. col 1 = AU01_r). Each row represents one frame of a slice. Row idx is a multiindex with entries [subj_id, slice_id, label, glob_idx].
features(pd.DataFrame) Dataframe with feature data of all slices. Columns represent channels and their features. Channel index is a multiindex where index-level 0 represents the channel and index-level 1 represents the corresponding features. Each row represents one slice. Row index is a mutliindex with entries [subj_id slice_id label glob_idx_start glob_idx_end].
